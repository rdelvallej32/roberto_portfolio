CatchUp

Contact Management Application

Full stack web app built for users to add and manage their professional contacts.The purpose of the app is to help users keep track of the last time they contacted their professional contacts. Users have CRUD capabilities to interact with their contact management system, which is stored on the back end with a PostgreSql database.

Random Topic Generator Application

Full stack web application that helps users discover new topics to use in day-to-day conversations. Purpose of the app is to help users, who would like to learn new ways to start a conversation with other people. Users have CRUD capabilities to interact with the ratings data, which is seeded on the back-end in a PostgreSQL database and can receive current events from the New York Times API.
